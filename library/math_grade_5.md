# Grade 5 Mathematics: Fractions, Volume, and Coordinates
*Context: Mathematics, Grade 5, Arithmetic, Geometry, Logic*

## 1. Operations with Fractions and Decimals
- We can add and subtract fractions with different denominators by finding a common denominator.
- Multiplying fractions involves multiplying the numerators and denominators.
- Dividing fractions often involves using the reciprocal.
- We can multiply and divide decimals using standard algorithms.

## 2. Geometry and Volume
- 'Volume' is the amount of space an object occupies.
- Volume = Length x Width x Height.
- Volume is measured in 'Cubic Units'.
- Two-dimensional shapes have area; three-dimensional shapes have volume.
- Attributes of shapes help in their classification.

## 3. The Coordinate Plane
- A 'Coordinate Plane' is a 2D surface used for graphing.
- The 'X-axis' is the horizontal line; the 'Y-axis' is the vertical line.
- An 'Ordered Pair' (x, y) tells us exactly where a point is.
- The 'Origin' is the point (0, 0) where the axes meet.

## 4. Mathematical Patterns and Logic
- Numerical patterns follow rules (e.g., Add 3 to each previous number).
- Parentheses () show which part of an equation to solve first.
- Order of Operations: Parentheses, Exponents, Multiplication/Division, Addition/Subtraction.
- Logic helps determine if a mathematical statement is true or false.
