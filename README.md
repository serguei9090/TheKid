# The Kid (Ali) - Neuro-Symbolic Agent

## This is a personal learning project created for educational purposes and to explore different code concepts with ai concepts. 

* **Status:** Personal sandbox / Portfolio piece.
* **License:** This project is open-source and available for public educational use under the MIT License.
* **Purpose:** Academic research and technical skill development.

Welcome to Ali's repository. This project aims to build a robust, superhuman AI agent utilizing strict biological and neuro-symbolic principles to avoid "Contextual Drift" and semantic hallucination. 

## Documentation
Please refer to the `docs/` folder for comprehensive information regarding the core architecture, the technology stack, and recent upgrades. 

- **[Phase 20 Architecture & Tech Stack](./docs/architecture_phase_20.md)**: A detailed breakdown of the Prefrontal Cortex (PFC), Math logic validation via SymPy, recursive directory reading, and the tech stack used in development (Python, SQLite, `uv`, `ruff`).

## Quick Start
1. Place source material (PDFs/TXT/MD) anywhere inside the `library/` folder.
2. Run `python main.py` to initiate the background learning engine and populate the database cleanly.
3. Chat with Ali using `python chat_terminal.py`.
